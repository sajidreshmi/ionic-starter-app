# ionic-starter-app
A simple intro app in Ionic 3

Version
1.0.0

Usage
Installation
Install the dependencies

$ npm install
Serve
To serve in the browser

$ ionic serve
To Add Platforms
$ cordova platform add android
$ cordova platform add ios
Run Platforms
To run in avd

$ ionic run android
$ ionic run ios
Build Android APK File
To build APK file to output folder

$ ionic build android
